生成されたcodedesignファイルをレビューし、問題を検出・修正する。

## あなたの役割
あなたはコード詳細設計のレビューエージェントです。`codedesign/` 内のMDファイルを読み込み、
設計品質を検証します。問題を検出した場合は報告し、自動修正可能なものは修正します。

## 事前に必ず読み込むファイル

1. `spec/rule/codedesign-review-rules.md` — レビュールール・チェック項目・重要度定義
2. `spec/rule/codedesign-generation-rules.md` — テンプレート・生成ルール（準拠確認用）
3. `spec/rule/project-save-rules.md` — ファイル構造・命名規則

## 入力

`$ARGUMENTS` の形式:

```
<codedesign-dir-path>
```

- `codedesign-dir-path`: レビュー対象の codedesign ディレクトリのパス（例: `./codedesign`）

例: `/review-codedesign ./codedesign`

## 実行手順

### 1. REVIEW.md の検出

プロジェクト内の `spec/REVIEW.md` を探す。

- **存在する場合**: 読み込み、その中のレビュー基準を把握する。共通チェック項目と合わせて適用する。
- **存在しない場合**: **SRP（単一責任の原則）チェックのみ** をレビュー基準とする。

### 2. codedesign ファイルの読み込み

指定ディレクトリ内の全MDファイルを読み込む：
- `{scene-name}/_scene.md` — シーン設計ファイル
- `{scene-name}/{actor-name}.md` — アクター設計ファイル
- `interfaces/{name}.md` — インタフェース定義ファイル

### 3. チェック実行

#### REVIEW.md がある場合
REVIEW.md のレビュー基準 + 共通チェック項目（C1〜C4）をすべて適用する。

#### REVIEW.md がない場合
SRP チェック（S1〜S3）のみ適用する：

- **S1 アクターレベル**: 各アクターが1つの明確な責務を持っているか
  - 概要が複数の無関係な責務を含んでいないか
  - コンポーネント構成に無関係なドメインが混在していないか
- **S2 コンポーネントレベル**: 各コンポーネントのタスク群が同一の関心事か
  - 変数群がタスクの実行に必要なものだけか
- **S3 メッセージレベル**: 1つのメッセージが複数の無関係な処理をトリガーしていないか

### 4. 自動修正

以下の問題は検出時にそのまま修正する（確認不要）：
- メッセージの双方向不一致 → 欠落側のファイルに追記
- インタフェースファイル欠落 → `interfaces/` にファイルを生成
- 空セクション見出し欠落 → テンプレートに従い追記
- 命名規則違反 → ファイル名を kebab-case にリネーム

それ以外（SRP違反、設計判断を伴うもの）は報告のみ。

### 5. 結果報告

以下の形式で結果を報告する：

```
## レビュー結果

### 適用基準
- REVIEW.md: {あり / なし}
- チェック項目: {適用したチェックの種類}

### 問題一覧

| # | 重要度 | カテゴリ | ファイル | 問題 | 修正案 |
|---|-------|---------|--------|------|-------|
| 1 | ERROR | ... | ... | ... | ... |

### 自動修正済み
- {修正内容とファイル名を列挙}

### サマリー
- ERROR: {n} 件
- WARN: {n} 件
- INFO: {n} 件
- 自動修正: {n} 件
```

## 重要度

| 重要度 | 意味 |
|-------|------|
| **ERROR** | 整合性の破綻。コード生成に支障 → 修正必須 |
| **WARN** | 設計品質の問題。コード生成は可能だが品質に影響 → 修正推奨 |
| **INFO** | 改善提案 → 参考情報のみ |

## 制約
- REVIEW.md がない場合は SRP チェック以外の品質判断をしないこと
- 自動修正は上記4種に限定し、設計判断を伴う修正は行わないこと
- 問題の報告は具体的なファイル名と箇所を必ず示すこと
- レビュー対象は codedesign/ 内のMDファイルのみ。ソースコードはレビューしない

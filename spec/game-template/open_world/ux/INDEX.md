# open_world / ux

## プレイヤー体験

- 「気になるもの」 を視認して向かう **目線駆動の探索**
- メインクエストと並行する **道草の自由**
- 動的 (天候 / 時間 / NPC) との **創発的遭遇**
- 自分だけの体験 = **SNS / 配信での共有** 動機

## ゲーム体験

- メインクリア: 30-60h、 100% 達成: 100-200h
- ファストトラベル + 騎乗 で **広大さ** と **テンポ** の両立
- 大型アップデート / DLC で**継続性**
- ストーリー切替えタイミング (章) で**体験の山場**

## NG パターン

- スカイボックスばかりで **中身がない** → 空虚なマップ
- POI 多すぎ + 内容軽い → 作業化
- ロード地獄 (頻繁に loading 5-10s) → 探索意欲喪失
- セーブが任意のみ → 死亡時のロス苦痛

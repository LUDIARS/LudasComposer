# fighting / ux

## プレイヤー体験

- 1 試合 30-90 秒の **濃密** な読み合い
- コンボ完走の **流れるような気持ち良さ**
- 投げ抜け / 確反の **判断が結果に直結**
- ロールバックネットコードで **オンラインも体感は対面**

## ゲーム体験

- ストーリーモード + 対 CPU + ローカル対戦 + オンライン
- トレモで **フレーム可視化** + コンボ練習
- リプレイ + フレームステップで **解析の楽しみ**
- 大会向け演出 (キャラセレ + ステージ + 観戦モード)

## NG パターン

- ロールバック非対応 → オンライン体験が崩壊
- 入力遅延 (内部 + ネット合計 > 100ms) → 操作不能感
- フレームデータ非公開 → トレモで自分で計測する苦行
- 起き攻め一択 (詰み状況) → 上手 vs 普通の格差固定

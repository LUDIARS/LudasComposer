# puzzle_casual / ux

## プレイヤー体験

- 1 ステージ 1-3 分の **すきま時間消化**
- 大連鎖時の **派手な爆発演出** で気持ち良さ
- ステージ目標達成 → **アニメ + ご褒美** で達成感
- 失敗時の **「あと一手だった...」** 残念感が再挑戦動機

## ゲーム体験

- ライフ 5 / 30 分回復 → **連続プレイは課金に誘導**
- ステージ難度は **試行回数 4-6 回でクリア** に調整
- 数千ステージ用意 → **生涯遊べる量**
- リフォーム / 物語進行で **メタ動機** (ステージのごほうび)

## NG パターン

- 課金圧が強すぎ → 信用低下
- アド表示頻度高すぎ → アンインストール
- ステージ難度が壁 (突破率 < 20%) → 大量離脱
- 連鎖演出が短い → 達成感が薄い

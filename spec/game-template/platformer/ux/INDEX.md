# platformer / ux

## プレイヤー体験

- ジャンプの **可変高さ** + **coyote time** が 「思い通りに飛べる」 感
- 落ちる → 即リスポーン → 「もう一度」 という再挑戦欲
- ステージ前進で **新ギミック** を学習する楽しさ
- ボス戦は短く明快 (パターン読み + ジャンプ判断)

## ゲーム体験

- 1 ステージ 1-3 分の短セッション設計
- ワールドごとに 「テーマ」 (氷 / 砂漠 / 城) と 「新ギミック 1-2 個」
- ボーナスステージ + 隠しステージで **横道の楽しさ**
- スピードラン愛好者向けに **タイマー表示** + ベスト記録

## NG パターン

- ジャンプ精度がピクセルパーフェクト要求すぎ → 苛立ち
- リスポーン位置が遠い → 諦観
- カメラが追いつかず先が見えない → 理不尽な死

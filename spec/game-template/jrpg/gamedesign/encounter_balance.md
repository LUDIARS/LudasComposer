# エンカウンター頻度と戦闘時間

## エンカウント頻度

| 場所 | 頻度 | 形式 |
|------|------|------|
| 街中 | 0 (なし) | — |
| 街道 | 60-120s に 1 回 | ランダム or シンボル |
| ダンジョン (浅) | 30-60s に 1 回 | シンボル推奨 |
| ダンジョン (深) | 20-40s に 1 回 | シンボル + 強制戦闘部屋 |
| ボス前 | 専用部屋 | 固定 |

## 1 戦闘の所要時間

- 雑魚戦: 30-90s (1-3 ターン)
- 中ボス: 90-180s (5-10 ターン)
- ボス: 180-360s (10-20 ターン)
- 隠しボス: 5-15 分 (20-50 ターン)

## 設計原則

- **3 連戦で 1 回回復 (アイテム / 帰還) を要求**
  - リソース管理を発生させる
- **エンカウント率はステージごとに調整**
  - 同じスキル / 戦術が連発で飽きないように
- **シンボルエンカウントは可能なら採用**
  - プレイヤーが避ける選択肢を持つ → 自由感
  - 戦闘の発生に意図が感じられる
- **ボス前に必ず回復ポイントを置く**
  - 「ボス覚悟して戦う」 という宣言

## ターン経済

- プレイヤーが 1 ターンで殺せる雑魚数 ≒ パーティ人数
  - 4 人 vs 4 体の雑魚で 1-2 ターン決着が標準
- 雑魚が複数行動する場合はパーティの**回復役**で応急処置可能な範囲

## 経験値設計

```
雑魚 1 体: Lv 比 1.0x で 必要 XP の 1/30 程度
ボス 1 体: 必要 XP の 1/3 程度
宝箱の経験値書: 必要 XP の 1/10 程度 (希少)
```

これにより、 通常進行で 1 章クリア = 5-10 Lv up になる。

## 状態異常の頻度

- 雑魚はレア (10-20%) に状態異常付与
- ボスの **特定攻撃** は状態異常確定 (回避手段がある)
- 状態異常は **3 ターン以内** に切れる、 アイテムで即解除可

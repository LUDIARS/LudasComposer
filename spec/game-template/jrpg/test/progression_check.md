# 進行 / セーブ自動回帰

## ユニット

- `BattleState`: ATB / 行動 / リザルトの分岐網羅
- `EventScript`: choose / set_flag / learn_skill 等の effect
- `Save`: snapshot → write → read → restore で全フィールド一致

## 統合

### 章 1 クリアシナリオ

```text
1. 新規ゲーム開始
2. プリセット入力ログを 「章 1 ボス撃破」 まで再生
3. 期待: 経験値 / 装備 / フラグが想定範囲
4. 各イベントが正しく発火
5. セーブ → ロード → 同等状態
```

### 章をまたいだ進行

```text
1. 章 1 終了状態をロード
2. 章 2 開始イベント → クエスト発生 → クリア
3. 期待: 章 1 のフラグが保持されつつ、 章 2 のフラグが追加
```

## バランス自動チェック

- 各章の **想定 XP / 通貨範囲** を define、 章クリア時にレンジ内
- ボス TTK が **5-15 分** 範囲内 (パーティ Lv 帯標準)

## 手動

- [ ] 全主要キャラのレベルアップ演出が見える
- [ ] 弱点 hit でダメージ表示 + UI フィードバック
- [ ] ボス専用 BGM が章ごと違う
- [ ] 任意セーブが街 / ダンジョン両方で動く
- [ ] 状態異常 UI で残ターン数が分かる

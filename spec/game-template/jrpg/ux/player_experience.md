# jrpg — プレイヤー体験

## 探索 → 戦闘 → 成長の体験ループ

```
街でイベント / 次の目的地を聞く
    ↓
道中 / ダンジョンを探索
    ├─ ランダム / シンボルエンカウント
    │   ├─ コマンド選択 (1 ターン 5-15s 程度)
    │   ├─ 弱点を試す → 効いた / 効かない
    │   ├─ 状態異常 / バフを掛ける
    │   └─ 撃破 → XP / 戦利品
    └─ 宝箱 / 隠し通路 / NPC 会話
    ↓
ボス戦
    ├─ パターン読み + 状態管理
    └─ 撃破 → 次の街 / 章
    ↓
レベルアップ + 装備更新
    ↓
新しい街 + 新しい物語
```

## 戦闘 1 ターンの体験

```
自分のターン到着 (ATB 満タン)
    ↓
プレイヤー: パーティ全員の行動を考える
    ├─ 物理攻撃 (確実 / 普通)
    ├─ 魔法 (弱点なら大ダメージ / MP コスト)
    ├─ アイテム (HP 回復 / 状態回復)
    ├─ 防御 (受けるダメージ半減)
    └─ 逃げる (低 Lv 戦の救済)
    ↓
コマンド送信 → アニメ → 結果
    ↓
敵ターン: 食らう or 回避
    ↓
HP 残量チェック → 次行動を考え直す
```

## 「成長を実感する」 仕掛け

- レベルアップ時の **派手な演出** (キャラ単位で SE + 文字)
- 新スキル習得時に **使ってみたくなるツールチップ** (「次の戦闘で試そう」)
- 宝箱の中身は **常に何か嬉しい** (空箱は無し、 ハズレでも消費アイテム)
- 装備変更で **見た目が変わる** (任意でも、 数アイテムは必ず) → 自分のキャラ感

## 「物語に入り込む」 仕掛け

- 章ごとの **明確な目的** (○○を探す、 ○○を倒す)
- 章末に **小さなクライマックス** + 次章への引き
- パーティキャラ間の **小さな会話イベント** (旅の合間に流れる)
- メインキャラの **過去の謎** を小出しにし、 終盤で解決

## NG パターン

| NG | 起きる原因 | 対策 |
|----|-----------|------|
| 「単調」 | 戦闘パターンが一辺倒 | 弱点 / 状態異常 / 連携で抑揚 |
| 「育てゲー疲労」 | XP テーブルが急峻すぎ | レベル上昇テンポを再設計 |
| 「物語が頭に入らない」 | テキスト早送りに重要情報 | 重要セリフは選択肢で確認 |
| 「迷子」 | クエストログが弱い | 目的地マーカ + 直近会話の再生 |
| 「強くてニューゲーム」 が無い | 達成後の動機ゼロ | 周回モードを必ず実装 |

# jrpg — ユビキタス言語

| 用語 | 説明 |
|------|------|
| ATB | Active Time Battle。 時間で行動順が来る方式 |
| CTB | Conditional Turn Battle。 行動順が予告される方式 |
| パーティ | 戦闘に参加する仲間集団 (3-4 人) |
| リザーブ | 戦闘に参加しない控えメンバー |
| ジョブ / クラス | キャラの役職。 ステ補正 + 習得スキル |
| MP / SP | 魔法 / スキルポイント。 戦闘中の消費リソース |
| 状態異常 | 毒 / 麻痺 / 沈黙 / 混乱など、 ターン経過で減衰する負効果 |
| 属性 | 火 / 水 / 風 / 土 / 雷 / 光 / 闇 など。 弱点 / 耐性 / 無効 / 吸収 |
| エンカウント | 戦闘発生のトリガ |
| ボスゲージ | フェーズ移行を示す HP 帯の表示 |
| やり込み | クリア後コンテンツ。 隠しボス / 裏ダンジョン |
| キーアイテム | 物語進行に必要な譲渡 / 売却不可アイテム |

## 命名ルール

- コードは **英語スネークケース**: `command_battle`, `stat_system`
- 仕様書は **日本語 + 英語併記**

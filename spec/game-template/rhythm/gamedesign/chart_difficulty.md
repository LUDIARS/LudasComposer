# 譜面難度設計

## レベル指標

- 数値: 1-13 (BMS / IIDX 流) もしくは 1-26 (DDR 流)
- ラベル: Easy / Normal / Hard / Expert / Master
- 内部メトリクス: 平均 NPS + ピーク NPS + 同時押し最大 + 種類使用率

## 難度別目安 (4 レーン譜面)

| Lv | 平均 NPS | ピーク NPS | ノート種 |
|----|---------|----------|---------|
| 1 | < 1 | < 2 | Tap のみ |
| 3 | 1-2 | 3-4 | Tap + Hold |
| 5 | 2-3 | 4-6 | + Slide 軽め |
| 7 | 3-5 | 6-8 | + Flick / 同時押し 2 |
| 9 | 5-7 | 8-10 | + 高密度連打 |
| 11 | 7-9 | 10-12 | + 複雑配置 / 同時押し 3 |
| 13 | 9+ | 12+ | 全部 / 鬼配置 |

## 譜面の作りかた (チェックリスト)

- [ ] 楽曲の **メイン旋律** に対応するノート配置
- [ ] サビ部分は **密度高く** 派手に
- [ ] 静かな部分は **休符 + 低密度**
- [ ] **休憩区間** (5-10s 何もしない) を曲中に最低 1 回
- [ ] 終曲ラスト 5-10s は **盛り上がり** 配置
- [ ] **見切れない / 視認限界以上の連打** を入れない (高難度を除く)
- [ ] **無理押し** (同時押し不可能配置) を避ける

## レーン配置の禁忌

- 同レーン連打が間隔 50ms 未満 (高難度の表現以外で)
- 端レーン → 反対端レーンの**遠距離急変** (人間工学的に不可能)
- 3 連同時押しを 10 連続 (不可能)

## エディタチェック機能

- 譜面読み込み時に **無理押し検出** + 警告表示
- 平均 / ピーク NPS の自動計算
- 推定難度レベルの提案

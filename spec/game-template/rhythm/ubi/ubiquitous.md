# rhythm — ユビキタス言語

| 用語 | 説明 |
|------|------|
| ノート / ノーツ (Note) | 譜面に並んだ叩く対象。 タップ / ホールド / スライド / フリック等 |
| 譜面 (Chart) | 楽曲のノート配置データ |
| BPM | Beats Per Minute |
| レーン (Lane) | 入力チャネル。 4-7 が一般的 |
| 判定窓 | PERFECT / GREAT / GOOD の許容 ms 範囲 |
| ハイスピード | 譜面のスクロール速度 (個人最適) |
| オフセット | 音 / 映像 / 入力の補正 ms |
| キャリブレーション | 環境ごとのオフセット自動測定 |
| フルコンボ (FC) | MISS 0 でクリア |
| パーフェクト全 (AP / PFC) | 全判定 PERFECT |
| ランク | S+/S/A/B/C/D 等の最終評価 |
| コンボボーナス | コンボ伸長時のスコア倍率 |
| 譜面難度 | Easy/Normal/Hard/Expert/Master 等のラベル |
| 譜面レベル | 数値による難度 (1-13 / 1-26 等) |
| NPS | Notes Per Second。 譜面密度の指標 |

## 命名

- コード: 英語スネーク (`note_chart`, `timing_judge`)
- 仕様書: 日本語 + 英語併記

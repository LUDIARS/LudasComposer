## 核

| 軸 | 狙い |
|----|------|
| **一体感** | 音楽 + 入力 + 視覚の同期した気持ち良さ |
| **習熟感** | 同じ曲を反復し、 PERFECT が増える快感 |
| **興奮** | コンボ + フルコンボ達成の高揚 |
| **多様性** | 多曲 + 多難度 + 多種ノートでの飽きさせなさ |
| **競争性** | スコア + ランキング (任意) |

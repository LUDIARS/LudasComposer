# rhythm — プレイヤー体験

## セッション開始 → プレイの体験

```
タイトル → 曲選択 (ジャンル / 難易度を聴き比べ)
    ↓
カウントダウン (3, 2, 1)
    ↓
ノーツが流れてくる
    ├─ Tap: 単打、 PERFECT で気持ち良い
    ├─ Hold: 押しっぱなし、 終端タイミング
    ├─ Slide: スワイプ
    └─ Flick: 弾く
    ↓
連続成功でコンボ + 派手 UI
    ↓
ミスで FX が暗転 + コンボ切断
    ↓
楽曲終了 → スコア / ランク / 判定分布
    ↓
リザルト画面 → 再挑戦 / 別曲
```

## 1 ノートの体験 (ms 単位)

```
ノート出現 (画面外) → 流れてくる (1.5-3s)
判定窓 ±120ms 以内に押下
    ↓
PERFECT → 派手な ✨ + 高音 SE + コンボ +1
GREAT  → 中規模 + 中音 SE + コンボ +1
GOOD   → 控えめ + 低音 SE + コンボ +1 (一部設定で切断)
MISS   → 暗 FX + 低音衝撃音 + コンボ切断
```

## 「上手くなった」 と感じる仕掛け

- **判定分布グラフ** (リザルト画面) — 早 / 遅の偏りが見えると意識して直せる
- **ベストスコア更新時** の派手な祝福
- **フルコンボ達成** の **特別演出** (キャラ / ロゴ表示)
- **練習モード** (難所部分繰返し)
- **ハイスピード設定** で視認性をプレイヤーが調整可

## NG パターン

| NG | 起きる原因 | 対策 |
|----|-----------|------|
| 「ズレてる」 (合わせる感がない) | audio offset 校正なし | 起動時自動キャリブ + 手動 |
| 「叩いても判定されない」 | 入力レイテンシ大 / バッファ無し | OS 直近 input + buffer |
| 「飽きる」 | 譜面 / 曲のバリエ少 | 段階的解禁、 コミュニティ作譜 |
| 「視認できない」 | スピード遅すぎ / 速すぎ | ハイスピ設定 + 流れ方向 (上 / 下 / 円) を選択可 |
| 「失敗が罰すぎる」 | フルコンが厳しすぎ + 報酬遠い | "S ランク" を 95% 程度に、 ミス可枠を確保 |

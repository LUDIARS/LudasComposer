# deckbuilder_roguelike / ux

## プレイヤー体験

- カードを引いた瞬間に **「この手番をどう料理するか」** の悩み
- レリック取得で **ビルドが固まる** 瞬間の高揚感
- ボス戦で **完璧な手** が決まる気持ち良さ
- 死亡しても新カード / 新キャラ解禁 = **常に成長**

## ゲーム体験

- 1 ラン 60-120 分
- 章 (Act) ごとにエレベーター → ボス
- 通常戦 / エリート / ボス / 焚火 / 商人 / 不思議イベント のノードバラエティ
- アンロック解放で 50-100 ランで全要素到達
- リプレイは **シナジー発見** + **記録更新**

## NG パターン

- カード説明が長文 / 不明瞭 → 学習コスト過大
- 引きが極端な運ゲー → 戦略の影響薄
- レリック効果が分かりづらい → 価値判断不能
- アンロック条件が固すぎ → 中盤離脱

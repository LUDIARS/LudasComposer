# hack_and_slash / ux

## プレイヤー体験

- 「**画面が埋まる敵 + 派手 AOE + 大量パーティクル**」 の爆発感
- アイテムドロップの **連続的小報酬**
- レアドロップの **小さな祝祭** (黄金テキスト + SE)
- ビルド完成度を上げて **同じステージを高速周回**

## ゲーム体験

- メインキャンペーン: 20-40 時間
- 「マップ廻し」 でレベル上げ + 装備集め: 100-1000+ 時間
- 季節制で **環境変化** + ビルドメタの更新
- 友人とのマルチで **倍速進行 + おすそ分け**

## NG パターン

- アイテム差異が薄い → 拾う動機薄
- スキルツリーが線形 → ビルド多様性ゼロ
- ストレージ / ソートが弱い → アイテム探しに時間溶ける
- エンドゲームが弱い → クリア後即離脱

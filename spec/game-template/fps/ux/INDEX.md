# fps / ux

## プレイヤー体験

- エイム精度の **個人技** がスコアに直結
- 足音 / 銃声から **敵位置を読む** 環境認識
- リロード / 切替 / グレネードの **状況選択**
- キル / デスの瞬間が短く強い感情起伏

## ゲーム体験

- 1 試合 15-40 分
- ランクマッチ + アンランク + カスタム + イベント
- 大型アップデート / シーズンで **武器バランス + マップローテーション**
- e スポーツ / 観戦モード = **継続性源泉**

## NG パターン

- ネット遅延 (ping > 80ms) → 撃ち負けが運になる
- マップが偏った (片チーム圧倒的有利) → 試合の半数が無意味
- アンチチート甘い → ゲームの根幹崩壊
- 武器バランスがメタ硬直 → 武器選択の意味喪失
- VC 罵倒の温床放置 → 新規 / 女性プレイヤー離脱

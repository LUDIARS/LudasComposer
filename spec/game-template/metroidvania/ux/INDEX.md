# metroidvania / ux

## プレイヤー体験

- 行き止まりに当たって 「あれ?」 → 後で能力獲得 → **「ここだ!」** の発見快感
- マップを埋める **探索ゲーム性**
- ボス撃破で **能力獲得** = 即時実用
- 100% コンプリートを目指す **二度味わい**

## ゲーム体験

- 8-15 時間でメインクリア
- 100% コンプは 20-40 時間
- 探索ヒントは **ヒント残しすぎず、 隠しすぎず**
- 後半に取得能力で序盤エリアの **隠し領域** が開く

## NG パターン

- 進行不能なバグ (能力なしで通れる場所) → 致命的
- マップ標示が薄い → 迷子地獄
- 隠し要素無し → 100% コンプの動機ゼロ
- ボスが運ゲー / 雑 → 能力獲得の意義喪失

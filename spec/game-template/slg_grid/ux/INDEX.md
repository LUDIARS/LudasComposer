# slg_grid / ux

## プレイヤー体験

- 1 ターンに **何分でも考えられる** 静寂の戦略
- 全行動の **結果予測** が見えるため理不尽は無い
- ユニット死亡が **永久** (FE 系) なら緊張感が爆増
- 章クリアで **物語進行** + 新ユニット加入

## ゲーム体験

- 1 章 = 1 ステージ = 30-90 分
- 章間に**準備フェーズ** (装備 / 配置 / 仲間会話)
- ボス章 / 撤退章で **特殊勝利条件** が緊張曲線
- クリア後に **ハードモード** + **支援会話コンプ**

## NG パターン

- 戦闘予測が不正確 (実乱数で大きくブレる) → 「予測の意味」 喪失
- 詰む状況がある (リセット必須) → 任意セーブを章内ターン頭で許容
- 仲間ユニットの個性が薄い → 死亡してもダメージなし → 永久死の意義喪失

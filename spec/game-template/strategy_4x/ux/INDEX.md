# strategy_4x / ux

## プレイヤー体験

- 「もう 1 ターンだけ...」 で 6-20 時間が溶ける長尺セッション
- 自国の文明を **育てる** という長期育成感
- 多軸勝利で **自分だけの勝ち筋**
- AI 文明との **外交ドラマ** (裏切り / 同盟 / 報復)

## ゲーム体験

- 序盤: 探索 + 開拓 (1-3h)
- 中盤: 戦争 / 同盟 / 科学レース (3-10h)
- 終盤: 勝利条件への駆込み (3-5h)
- アラート / 通知が **「やるべき次の行動」** を明示 (情報過多になりがちなため必須)

## NG パターン

- ターン進行が遅い (AI 思考時間長い) → だれる
- 通知が多すぎ → 重要情報を見逃す
- 序盤と終盤で同じ操作 → ペース無し → 飽き
- セーブが分かりづらい (autosave 唯一など) → 大事故時にロス

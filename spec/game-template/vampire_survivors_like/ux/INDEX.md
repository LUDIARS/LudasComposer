# vampire_survivors_like / ux

## プレイヤー体験

- 操作は **移動だけ** = 片手プレイ可、 ながら見可
- レベルアップで **3 択選択** が 30 秒に 1 回 → 戦術が変わり続ける
- 数百体の敵を AOE で **薙ぎ払う爽快感**
- 30 分耐久 → 死亡 → **「次はあのビルドで」** の即再挑戦

## ゲーム体験

- 1 ラン = 20-30 分の使い切り設計
- メタ進行で **長期動機** を作る (永続パッシブ / アンロック)
- ビルド多様性 = **シナジー発見** が再プレイ動機
- ボス級敵 = 5-10 分間隔で出現、 ピーク作り

## NG パターン

- 進化武器の条件が分かりにくい → ビルドが偶然頼り
- メタ進行の **アンロック条件** が見えない → 何をすべきか不明
- 死亡時のロスが大きすぎ → 再挑戦の気力喪失 (1-2 分の準備時間以内に次へ)

# moba / ux

## プレイヤー体験

- ピックバンの **心理戦** から始まる
- 序盤レーン戦 → ジャングル / ガンク → チーム集結 → 最終戦闘 の **波**
- スキルショット命中の **個人技** + チーム連携の **協調**
- 試合 25-40 分で **濃密なドラマ**

## ゲーム体験

- 季節 (シーズン) 制で **ランクリセット** + 新メタ
- 月次 / 隔週パッチで **環境変化**
- 観戦モード + プロシーン = **eスポーツ寄与**
- マッチング待機が短いほど良い (< 90s 目標)

## NG パターン

- ヒーロー間バランスが極端 → メタ硬直 / 楽しみ偏り
- 入力遅延 (ping > 80ms) → 判定がランダム化
- マッチングが偏る (新規 vs 古参) → 新規離脱
- 罵倒チャット放置 → コミュニティ毒化 → 離脱

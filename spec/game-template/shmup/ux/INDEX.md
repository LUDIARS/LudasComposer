# shmup / ux

## プレイヤー体験

- 数百の弾を **避ける** = 究極の集中
- 集中モード (低速移動) で **小 hitbox を細やかに動かす**
- グレイズ (紙一重回避) で **加点** = リスクテイク
- ボス戦の長尺パターン読み + 撃破

## ゲーム体験

- 1 ステージ 3-5 分 × 5-7 ステージ
- 1cc クリア (= ノーコンティニュークリア) が中上級プレイヤー目標
- ノーミス / フルチェイン / リプレイ提出のスコアアタック文化
- パワーアップシステムで残機温存とリスクテイクのバランス

## NG パターン

- 弾速が極端 (速すぎ / 遅すぎ) で避けられない / だれる
- 自機 hitbox が見た目通り (大きすぎ) → 「擦って当たる」 不当感
- ボムの強さがピーキー (使う / 使わないで二択化)
- リプレイ非対応 → スコア競争コミュニティ形成不能

# ボス戦設計

## チェックリスト

ボス 1 体ごとに必ず満たす:

- [ ] 最低 2 フェーズ (HP しきい値で遷移)
- [ ] フェーズ移行時に **専用演出** (BGM 切替 / カットイン)
- [ ] 各フェーズ 4-7 種の攻撃パターン
- [ ] 各パターンに **0.5-1.0s の予兆** (姿勢 / VFX / SE)
- [ ] パリィ可能技を最低 1 つ含む (体幹崩しの動機)
- [ ] 確反窓を最低 1 つ含む (攻撃の動機)
- [ ] 直近で死亡した場合の **デモ動画 / ヒント** (任意)
- [ ] BGM が独自 (ザコ戦と区別)
- [ ] 撃破時の **特別演出** (ヒットストップ + スロー + ズーム)
- [ ] 撃破時の **報酬** (新装備 / アイテム / アンロック)

## サンプル: 「焔の剣士」 (Lv 25 想定)

### 概要

- 火を纏う近接ボス。 動きは速いがリーチは中。 体幹崩し → 忍殺で大ダメージ可。

### Phase 1 (HP 100% → 50%)

- **基本 3 連撃** (左→右→重) — 3 連目はパリィ可、 確反窓 18f
- **突進斬り** (1.0s 予兆 + 距離 8m 突進) — 横歩きで回避
- **火炎ジャンプ攻撃** (1.5s 予兆) — タイミングで真下に潜り込んで回避

### Phase 移行

- HP 50% でいったん下がり、 剣に火を纏う (BGM 強奏)。 1.5s 演出。

### Phase 2 (HP 50% → 0%)

- Phase 1 の全パターン + 火力 1.3x
- **追加: 火炎波** (横一線、 ジャンプで回避)
- **追加: 連続斬り** (5 連、 4 連目以降にパリィ可)
- **追加: 自爆**気味の捨て身 (HP 20% 以下、 大ダメージだが大確反)

### 撃破時の演出

```
HP 0 → スロー 0.5x で 2.0s
       → 「○○を撃破した」 文字 + SE
       → 1.0s ホワイトアウト → 報酬画面
```

## ボスバランス目安

| プレイヤー L 別 | TTK (Time To Kill) 範囲 |
|---------------|--------------------------|
| 同 Lv 達者 | 60-90s |
| 同 Lv 平均 | 100-180s |
| やや低 Lv | 180-300s (粘れば勝てる) |
| 低 Lv | クリア困難 (装備の重要性を体験させる用) |

## ボスを作るときのワークフロー

1. **コンセプト 1 文** (例: 「火を纏う近接、 体幹崩しで大ダメージ可」)
2. **見た目モック** (シルエット → 動き)
3. **攻撃パターン仮定義** (4-7 個 / Phase)
4. **TOML / Excel に数値** (発生 / 持続 / 硬直 / ダメージ)
5. **AI スクリプト** (パターン選択ロジック)
6. **撃破演出仮置き**
7. **実装 → 内部試遊**
8. **TTK 計測** + バランス調整
9. **配信向け演出** チェック (映える瞬間が必ずある)

# action / ux — プレイヤー体験 / ゲーム体験

世界観やストーリーは個別タイトルで決めるためここでは記述しない。
このジャンル (action) で **プレイヤーに与えたい体験 (UX)** と **ゲーム全体の体験設計** を記述する。

## 詳細ページ

- [player_experience.md](player_experience.md) — プレイヤー視点で何が起きるか / 何を感じるか
- [game_experience.md](game_experience.md) — ゲーム側がどんな体験を組み立てているか (波 / 山 / 落下)

## このジャンルが目指す核

| 軸 | 狙い |
|----|------|
| **緊張感** | 一瞬の判断ミスが死につながる手応え |
| **習熟感** | 死を繰り返すほど上手くなる感覚 |
| **打撃感** | 一撃を「重い」 と感じさせる演出 |
| **読み合い** | 敵の予兆を読んで対応する駆け引き |
| **挑戦欲** | ボスや高難度に再挑戦する意欲 |
